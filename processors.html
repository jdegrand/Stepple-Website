Analytics Processors

Last updated: April 27, 2026

This page lists the third-party data processors currently used to handle the anonymous app-usage, engagement, and diagnostic data described in Section 7 of the Privacy Policy.

Current Processors

  • PostHog — product analytics. Receives anonymous events about how the app is used (screen views, feature usage, onboarding progress, paywall interactions, subscription lifecycle events, retention milestones, error and performance diagnostics). Each event is keyed to a random identifier generated on your device. No health or fitness values are ever sent. No pet names, leaderboard names, or other free-text user input are ever sent. No name, email, Apple ID, or iCloud user ID are ever sent. IP-based geolocation is disabled. Data is hosted in the United States. PostHog acts as a data processor under the developer's instructions. See posthog.com/privacy for PostHog's privacy practices.

Subscription lifecycle data from Apple's App Store Server Notifications service is processed by the developer directly on the developer's own server and is not shared with any third party.

Changes to This List

This page will be updated whenever processors are added, removed, or changed. The categories of data collected, the anonymous identifier scheme, and the commitments described in the Privacy Policy do not change when this list does.
